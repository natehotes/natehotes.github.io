Welcome to my svg prodject, I'm going to explain to you how this prodject works. The charts are broken into country, names, values, and chart headers and classed as symbols to allow for to allow them to be easily manipulated (for each chart country values and headers change but country names stay the same). What is left after that is a mess of all three charts being visibile, the charts needed to be selectivley visible. Next I used infographic rules and clarified the three charts as large,small, or medium to embed two of the headers and values, making them invisible, but filling out country names and the header and data of the chart that I wanted to be visible. Laslty I wanted to make it so, as the screen expanded, the other charts appeared and dissapeared so that the viewer can view all three easily. To do this I used a skeletal vector graphic, ot svg file, because I wanted a chart to appear from 0 to 400 px, width a second one from 400 to 525 px width, and lastly one to apprear above 525 px. To do this I set each chart header and data with a maximum width except the last chart where I used display: none to keep it invisible before 525.
